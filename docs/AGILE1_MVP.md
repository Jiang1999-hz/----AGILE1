# Agile 1 MVP 定义

## 当前目标

先做一个能给小私塾老板和老师现场演示的 Demo，用最短时间说明这套产品能帮助他们降本增效。

## Agile 1 必须做的功能

### P0

1. 学生列表
- 展示学生基本情况、当前得分、到课率、风险等级。

2. 学生详情
- 展示学习目标、能力图、近期进度、家长沟通摘要。

3. 在线做题
- 学生可以在网页上直接做题。
- 提交后系统自动评分。

4. 做题结果联动
- 做题结果自动进入学习进度。
- 做题结果自动影响能力诊断和风险判断。

5. AI 辅导建议
- 根据学生表现生成老师下一步辅导建议。

6. 家长反馈摘要
- 把系统结论转成更容易给家长看的说明。

## 这版 Demo 主要打的痛点

1. 老师批改和整理学生情况很费时间。
2. 学生做题数据没有变成可复用的诊断信息。
3. 家长看不到“孩子具体哪里卡住、哪里进步”。
4. 塾长很难提前发现风险学生。

## Agile 1 验收标准

- 可以切换学生。
- 可以在线做题并提交。
- 提交后可以看到分数和反馈。
- 学生画像与进度会发生变化。
- 可以把这个流程完整演示给潜在客户。
