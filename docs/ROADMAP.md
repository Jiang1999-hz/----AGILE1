# 产品迭代路线图

## Agile 1

主题: `AI 学习进度 + 能力诊断`

交付内容:
- 学生列表
- 学生详情
- 能力图
- AI 诊断建议
- 家长反馈摘要
- 本地演示版

## Agile 2

主题: `真实数据录入`

建议交付:
- 学生成绩录入
- 出勤录入
- 教师备注录入
- 基础后台数据存储
- 可新增和编辑学生

## Agile 3

主题: `真实 AI 接入`

建议交付:
- 接入 OpenAI API
- 根据学生数据动态生成诊断建议
- 动态生成家长反馈
- 动态生成补强题建议

## Agile 4

主题: `私塾经营与留存`

建议交付:
- 风险学生面板
- 面谈记录
- 家长跟进记录
- 续费风险预测
- 简单经营仪表盘

## Agile 5

主题: `交付闭环`

建议交付:
- 报告导出
- LINE / 邮件通知
- 多老师协作
- 权限管理
- 校区视图
