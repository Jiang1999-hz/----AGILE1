# Juku AI Cloud Demo

一个面向日本小私塾的本地 Demo。

## 当前演示主线

1. 查看学生列表
2. 进入学生画像
3. 学生在线做题
4. 系统自动评分
5. 自动更新学习进度与能力诊断
6. 给出老师建议和家长反馈

## 本地运行

```powershell
node .\server.js
```

浏览器打开：

```text
http://localhost:4173
```

## 当前适合拿去讲的价值

- 帮老师省去批改和整理反馈的时间
- 把学生做题行为沉淀成进度数据
- 帮家长看懂学生哪里进步、哪里薄弱
- 帮塾长更早发现需要跟进的学生
